About Signal K Mobile
Signal K Mobile is an open source mobile application developed by Ilker Temir for leveraging and promoting Signal K, a modern and open data format for marine use.
It has been developed to promote the usage of Signal K and is NOT RECOMMENDED for navigation.
If you do not have any Signal K sensors, check BoatSensord project on GitHub for a list of Signal K capable DIY sensors.
Source code is available at GitHub under Apache License version 2.
Signal K Mobile is not affiliated with, nor is endorsed by the Signal K working group.
